# HOPE3 Foundation Feedback Form

A premium, responsive online feedback form built with HTML, CSS, and JavaScript. 

## Features
- Modern UI with glassmorphism and mesh gradients
- Mobile-responsive design
- Form validation
- Automated data submission to Google Apps Script (Google Sheets) via JSON POST
- Real-time interaction feedback
- Lucide Icons integration

## Deployment
This is a static site that can be hosted on GitHub Pages, Vercel, or Netlify.
